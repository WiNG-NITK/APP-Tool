APP-Tool
========

A ns2 tracefile Analyser.  

The application requires a Linux based environment and folowing libraries:

	Gtk+ for python: python-gi
	
	MatplotLib Library: python-matplotlib

To run:

	click GUI_Main.py
	
	or
	
	in terminal : python GUI_Main.py
